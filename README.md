# RJ Yanson - Premium Portfolio

A high-end, luxury developer portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Ultra-Modern Design**: Dark theme with cyan accents and luxury minimalist aesthetic.
- **Smooth Animations**: Powered by `motion/react` for scroll-triggered effects and micro-interactions.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Glassmorphism**: Premium card designs with subtle blur and border effects.
- **Parallax Hero**: Dynamic background with floating particles and smooth scrolling.

## Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS 4
- **Animations**: Framer Motion (`motion/react`)
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Sections

- **Hero**: Large name, title, and strong CTA.
- **About**: Professional bio and contact details.
- **Skills**: Categorized technical arsenal with modern icons.
- **Projects**: Featured work cards with live demos and GitHub links.
- **Contact**: Modern contact form and social links.
- **Floating CTA**: "Book a Call" button that appears on scroll.
